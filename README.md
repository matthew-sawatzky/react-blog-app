## Features

- User authentication
- Responsive design

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/react-blog-app.git
   ```
2. Navigate to the project directory:
   ```sh
   cd react-blog-app
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

## Usage

1. Start the backend server:
   ```sh
   cd dummy-backend
   npm start
   ```
2. Start the frontend application:
   ```sh
   cd blog-application
   npm run dev
   ```
